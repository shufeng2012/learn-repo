单调栈主要用于**往前/后查找第一个小于/大于它的数**

**性质：**

+ 保证栈内元素有序

+ 操作
  
  1. 将元素加入栈中
  
  2. 删除前面符合要求的元素 *（如大于、小于这个数）*

类型：`stack`（无STL模板）

**使用：**

```cpp
stack<int> stk;
//令a数组为输入的数组，n为数组长度
for(int i=1;i<=n;i++){
	//单调上升
	while(stk.top()>=a[i])
		stk.pop()
	stk.push(a[i]);
}
```

# T627178 【模板】单调栈

## 题目描述

给出项数为 $n$ 的整数数列 $a_1…a_n$。

请找到第 $i$ 个元素

之前第一个大于 $a_i$ 的元素的下标（下标从1开始）

之后第一个大于 $a_i$ 的元素的下标。

若不存在，输出 $0$。

## 输入格式

第一行一个正整数 $n$。

第二行 $n$ 个正整数 $a_1…a_n$。

## 输出格式

分为两行输出，第一行输出之前第一个大于 $a_i$个下标。第二行输出之后第一个大于 $a_i$ 的元素的下标。每个数据之间用空格隔开。

## 输入输出样例 #1

### 输入 #1

```
5
1 4 2 3 5
```

### 输出 #1

```
0 0 2 2 0
2 5 4 5 0
```

## 输入输出样例 #2

### 输入 #2

```
6
3 1 1 2 2 3

```

### 输出 #2

```
0 1 1 1 1 0
0 4 4 6 6 0
```

## 说明/提示

对于 $30\%$ 的数据，$n\le 100$；

对于 $60\%$ 的数据，$n\le 5\times 10^3$ ；

对于 $100\%$ 的数据，$n\le 5*10^6,a_i\le 10^9$ ；
