**思想：** 从0开始，依照现有的结论不断向结果推进

# 示意图

```text
[结果]    ^ 从
[中间]    | 下
[中间]    | 往
[开始]    | 上
```

# 示例

求斐波那契数列第`n`项

```cpp
#include<bits/stdc++.h>
using namespace std;
int n;
int a,b,c;
int main(){
    cin>>n;
    a=1;b=1;c=1;
    for(int i=3;i<=n;i++){
        c=a+b;
        a=b;b=c;
    }
    cout<<c;
    return 0;
}
```
