# P1223 排队接水

## 题目描述

有 $n$ 个人在一个水龙头前排队接水，假如每个人接水的时间为 $T_i$，请编程找出这 $n$ 个人排队的一种顺序，使得 $n$ 个人的平均等待时间最小。

一个人的等待时间不包括他的接水时间。

如果两个人接水的时间相同，编号更小的人应当排在前面。

## 输入格式

第一行为一个整数 $n$。

第二行 $n$ 个整数，第 $i$ 个整数 $T_i$ 表示第 $i$ 个人的接水时间 $T_i$。

## 输出格式

输出文件有两行，第一行为一种平均时间最短的排队顺序；第二行为这种排列方案下的平均等待时间（输出结果精确到小数点后两位）。

## 输入输出样例 #1

### 输入 #1

```
10 
56 12 1 99 1000 234 33 55 99 812
```

### 输出 #1

```
3 2 7 8 1 4 9 6 10 5
291.90
```

## 说明/提示

$1\le n \leq 1000$，$1\le t_i \leq 10^6$，不保证 $t_i$ 不重复。

# 代码

```cpp
#include<bits/stdc++.h>
#define ll long long
using namespace std;
struct node{
	ll n,t;
};
ll n;
node t[1005];
ll ans;
int main(){
	ios::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);
	cin>>n;
	for(ll i=1;i<=n;i++){
		cin>>t[i].t;
		t[i].n=i;
	}
	sort(t+1,t+n+1,[](node a,node b){
		return a.t<b.t;
	});
	for(ll i=1;i<=n;i++){
		ans+=(n-i)*t[i].t;
		cout<<t[i].n<<' ';
	}
	cout<<endl;
	printf("%.2f",ans*1.0/n);
	return 0;
}
```
