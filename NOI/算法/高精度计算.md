# 加法
按照加法竖式计算
1. 将目标数倒过来
2. 按照加法运算进行运算
3. 将数倒过来

```cpp
string add(string a,string b){
	reverse(a.begin(),a.end());
	reverse(b.begin(),b.end());
	string c;
	int i=0,x=0;
	while((i<=a.size())||(i<=b.size())){
		c[i]=(a[i]-'0')+(b[i]-'0')+x;
		if(c[i]>=10){
			x=c[i]%10;
			c[i]%=10;
		}
		c[i]+='0';
	}
	if(x!=0){
		char t=x+'0';
		c+=t;
	}
	reverse(c.begin(),c.end());
	return c;
}
```
