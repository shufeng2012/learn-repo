**思想：** 从结果逆推，一步步分解并解决，*分治思想*

# 示意图

```text
[结果]    先|  ^上
[中间]    往|  |往
[中间]    下|  |后
[开始]      ---
```

# 示例

求斐波那契数列第`n`项

```cpp
#include<bits/stdc++.h>
using namespace std;
int solve(int n){
    if(n==0) return 0;
    if(n==1) return 1;
    return solve(n-1)+solve(n-2);
}
int n;
int main(){
    cin>>n;
    cout<<solve(n);
    return 0;
}
```
