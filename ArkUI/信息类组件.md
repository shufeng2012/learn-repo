# 信息展示类
## 文字
文字展示使用`Text`组件
```typescript
Text("message")
```
## 图片
图片展示使用`Image`组件
```typescript
//1.使用网络图片
Image('url')
//2.使用本地图片
//app指代resources/文件夹
Image($r('app.media.url'))
//3.使用系统图片
Image($r('sys.media.url'))
```
## 加载动效
加载动效使用`LoadingProgress`组件
```typescript
LoadingProgress()
```
## 二维码
二维码使用`QRCode`组件
```typescript
QRCode('url/message')
```

# 信息选择类
## 单选框
单选框使用`Radio`组件
```typescript
Radio({
	value: "value", 
	group: "group"
})
```
## 多选框
多选框使用`Checkbox`组件
```typescript
//圆形多选框
Checkbox()
//方形多选框
Checkbox().shape(CheckBoxShape.ROUND_SQUARE=1)
```
## 切换组件
切换组件使用`Toggle`组件
```typescript
Toggle({
	isOn: true   //是否开启
	type: ToggleType.Switch  //形状
})
```
## 下拉列表
下拉列表使用`Select`组件
```typescript
Select([
	//采用数组形式
	{value: "message1"},
	{value: "message2"}
])
```
