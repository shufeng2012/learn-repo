# 时间选择器
分为**日期选择器**`DatePicker`和**时间选择器**`TimePicker`
```typescript
DatePicker()
TimePicker()
```
# 信息标记
类似于微信的未读消息上面的
使用`Badge`组件
```typescript
Badge({
	count: 1   //信息标记数量
	style: {}  //可为空
}){
	Text("message")   //要被标记的组件
}
```
# 打分（五星好评）
打分使用`Rating`组件
```typescript
Rating()
```
# 手势密码锁
手势密码锁用`PatternLock`组件
```typescript
PatternLock()
```
# 列表（其实就是上下滚动）
列表使用`List`组件
列表项用`ListItem`组件
```typescript
List(){
	ListItem(){
		Text("message1")
	}
	ListItem(){
		Text("message1")
	}
}
```
# 轮播（广子用的）
可以手动滑动和自动轮播
轮播使用`Swiper`组件
```typescript
Swiper()
```

# 滚动
正宗的滚动，没有任何特效
滚动使用`Scroll`组件
```typescript
Scroll(){
	Text("message")
}
```
> 只能有一个根节点

# 滑动
可以快捷跳转，比轮播上强，无自动轮播
滑动使用`Tabs`组件
```typescript
Tabs(){
	TabContent(){
		Text("message1")
	}
	TabContent(){
		Text("message2")
	}
}
```
